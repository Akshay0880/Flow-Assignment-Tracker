# FLOW - Assignment Tracker

FLOW is a web application that helps students organize and manage assignments efficiently.

## Features

- Google Authentication
- Assignment Creation
- Assignment Tracking
- User Dashboard
- Assignment Status Management
- Dark Modern UI

## Technologies

- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Database
